Stacks

현재 저의 개발 능력은 이렇습니다.
현재에 머물러있지 않고 더 나은 개발자가 되기 위해 노력 중입니다.
기술 구분 로고 활용 수준 실력
HTML
웹 문서를 제작하는데 쓰이는 기본 프로그래밍 언어로써, HTML의 관한 전반적인 부분을 개발할 수 있습니다.
CSS
마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어로써, HTML에서 코딩한 코딩문을 디자인할 때 사용하며,
CSS를 이용해 하고자 하는 웹 디자인을 할 수 있습니다.
Android
휴대폰용 응용체제ㆍ미들웨어ㆍ응용프로그램을 한데 묶은 소프트웨어 플랫폼으로써, 안드로이드 휴대폰의 어플을 제작할 수 있습니다.
Python
컴퓨터 언어의 일종으로 간결하고 생산성 높은 프로그래밍 언어로써,Python을 이용해 인공지능 프로그램을 만들 수 있습니다.
Java
객체지향프로그래밍 언어로써, 보안성이 뛰어나며 네트워크 기능의 구현이 용이한 언어로써, 프로그래밍 중 가장 기본이 되는 언어이며,
Java를 활용하여 프로그래밍 할 수 있습니다.
JavaScript
객체 기반의 스크립트 프로그래밍 언어로써, 웹 브라우저 내에서 주로 사용되며, JavaScript를 이용하여 웹 프로그래밍을 할 수 있습니다.
Database
여러 사람에 의해 공유되어 사용될 목적으로 통합하여 관리되는 데이터의 집합로써, Oracle과 My SQL 등을 사용하여 Database를 구축할 수 있습니다.
Other 네트워크, 운영체제 등 컴퓨터 구조를 알고 있으며, 하이브리드 웹앱, 스프링, Firebase, C++, 자료구조를 알며 프로그래밍을 할 수 있습니다.
